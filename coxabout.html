Loot Sim


This is a tool I made to simulate loot drops.

Cox Kc: Simulate kills at Chambers of Xeric.
Cox Log: Simulate completing the Chambers Log.
Cox Masses: Simulate masses at Chambers. Multiple drops per raid possible if points above point cap (570,000).

Points: Points obtained per raid.
-Personal points for personal drops.
-Team points for team drops.
Kc: Simulate this number of raid completions.
Challenge mode: Simulate Challenge mode raid completions. Does not include point additions for making time, include this in your own points entry.
Display duplicates: Will display duplicate item drops. Unchecked will only display the first time you receive the drop with the exception being up to 3 twisted kits in challenge mode.
